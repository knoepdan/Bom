Basics: 
Host defines whether app runs development or production mode (etc)-> is defined by environment variable or cmd line args. (environment var can also be set by launch.setting.json)
Configuration is defined by: appsettings.json -> appsetting.{environment}.json (environment set by host), .... and others see docs.
